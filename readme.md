## About
[onigo-controller](https://github.com/comozilla/onigo-controller/) の server 部分です。  
sphero-websocket を使用しています。  
node.js サーバーで Bluetooth で通信し、信号を送っています。

## 準備

### モジュールをインストール
```
$ npm install
```

### モジュールをビルド
```
$ npm run build
```

### サーバーを起動
```
$ npm start
```
